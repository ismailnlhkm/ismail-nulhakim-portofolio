## What I Learned

1. **Berpikir Terstruktur dengan RTCC-O**  
   Saya belajar bagaimana memecah sebuah proyek menggunakan pendekatan terstruktur (Role, Task, Context, Constraints, Output). Pendekatan ini membantu mengurangi kebingungan saat pengembangan dan membuat proses perencanaan lebih jelas sebelum mulai menulis kode.

2. **Semantic HTML & Arsitektur CSS Modern**  
   Saya memahami pentingnya penggunaan semantic HTML untuk meningkatkan aksesibilitas dan keterbacaan struktur website. Selain itu, saya juga belajar menyusun CSS dengan pendekatan design system (menggunakan variable, scale spacing, dan komponen reusable) agar kode lebih rapi dan mudah dikembangkan.

3. **Kolaborasi Efektif dengan AI dalam Pengembangan**  
   Saya belajar bagaimana menggunakan AI sebagai alat bantu dalam proses pengembangan, bukan sebagai pengganti pemahaman. Dengan prompt yang lebih jelas dan iterasi yang tepat, AI dapat membantu mempercepat proses coding sekaligus meningkatkan kualitas hasil akhir.
